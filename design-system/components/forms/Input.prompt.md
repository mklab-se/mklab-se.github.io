Single-line text field. Inset well; focus lights an ember hairline + glow.

```jsx
<Input label="Work email" placeholder="you@firm.com" iconLeft={<Icon name="mail" size={16} />} />
<Input label="Passcode" error="That passcode has expired." />
```

Supports `label`, `hint`, `error`, `iconLeft/iconRight`, sizes `sm | md | lg`, and all native input props.
