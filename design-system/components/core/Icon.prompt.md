Renders a [Lucide](https://lucide.dev) icon at the brand's 1.5px stroke. Requires the Lucide UMD script on the page. Use for all UI glyphs — never hand-roll SVGs.

```jsx
<Icon name="arrow-up-right" size={20} />
```

`name` is any Lucide icon id. Defaults: `size={18}`, `strokeWidth={1.5}`, `color="currentColor"`.
