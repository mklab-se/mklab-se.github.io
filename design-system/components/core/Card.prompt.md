Surface container for grouped content. Default is a soft graphite panel; `glow` reserves the ember halo for the one card you want to pull the eye to.

```jsx
<Card interactive onClick={open}>
  <h3>Senior Advisory</h3>
  <p>For decisions that don't get a second attempt.</p>
</Card>
```

Variants: `default`, `inset` (well), `outline` (hairline only), `glow`. Padding `none | sm | md | lg`. Set `interactive` for hover lift.
